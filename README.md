# TOKUEN Web Application
The source code for the website interface of TOKUEN, utilising React + Vite as the frontend framework, Typescript, Node.js, TailwindCSS, and MaterialUI. TOKUEN is a fintech startup that aims to provide financial credit for Filipinos. By utilising blockchain technology, it guarantees that every transaction is secured, fast, and encrypted; and that the entire process is decentralised.

## What is TOKUEN?
TOKUEN is a blockchain-powered, credit-lending application that aims to provide financial credit to Filipinos. It is the flagship product of the company Artiphix, a startup founded by five computer science students of Adamson University under the mentorship and support of the Adamson University Neo Science and Technology Incubation Center (AdUNest).
